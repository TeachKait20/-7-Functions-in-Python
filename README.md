### –ï—â—ë –æ –ª–æ–∫–∞–ª—å–Ω—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö
–ë—ã–≤–∞–µ—Ç —Ç–∞–∫–æ–µ, —á—Ç–æ —Ñ—É–Ω–∫—Ü–∏–∏ –≤–ª–æ–∂–µ–Ω—ã. –ß—Ç–æ–±—ã –∏–∑–º–µ–Ω–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –∏–∑ –æ–¥–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ –≤ –¥—Ä—É–≥–æ–π, –∏—Å–ø–æ–ª—å–∑—É—é—Ç –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `nonlocal`. –û–Ω–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–±—Ä–∞—â–∞—Ç—å—Å—è –∫ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –∏–∑ –æ–±—ä–µ–º–ª—é—â–µ–π —Ñ—É–Ω–∫—Ü–∏–∏ –∏ –∏–∑–º–µ–Ω—è—Ç—å –µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –≤–Ω—É—Ç—Ä–∏ –≤–ª–æ–∂–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏.
